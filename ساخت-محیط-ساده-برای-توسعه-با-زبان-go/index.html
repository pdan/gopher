ساخت محیط ساده برای توسعه با زبان go

Jan 15, 2016 #آموزش #editor #شروع #نصب

داشتن یک محیط مناسب برای توسعه توسط یک زبان خاص همواره جزو مهمترین مسائل برای شروع کار با آن است. در اینجا سعی شده ابزار‌هایی برای ساخت محیط برنامه نویسی Go معرفی شود که در سه پلتفرم Linux, Mac, Windows قابل پیاده سازی باشند. همچنین تمامی مراحل در سیستم عامل Ubuntu 15.04 اجرا شده است.

نصب Go

برای نصب Go و مشتقاتش از چندین راه می‌توان وارد شد! که در اینجا دو روش را بررسی خواهیم کرد.

راه اول

در ابتدا آخرین نسخه را برای سیستم عامل مورد علاقه خود از صفحه رسمی golang دریافت و نصب کنید.

راه دوم (توصیه می‌شود)

معمولا مدیریت کنندگان نسخه (Version Managers) از طرفداران زیادی برخوردار هستند. برای زبان Go هم نرم‌افزار مدیریت نسخه خوبی ساخته شده با نام gvm که این قابلیت را به شما می‌دهد که نسخه‌های مختلفی از Go را روی سیستم خود داشته باشید تا در زمان نیاز به سرعت بین آنها سوییچ کنید. همانطور که می‌دانید غربی‌های خبیث این زبان را هم برای کشور عزیزمان تحریم کرده‌اند (مثلا اسمش نرم‌افزار آزاد هست) و gvm این امکان را به ما می‌دهد که بتوانیم Go را از منبع دیگری که چه از داخل و چه از خارج تحریم نباشد نصب کنیم.

ابتدا دستور زیر را در خط فرمان اجرا کنید تا بسته‌های پیش نیاز نصب شوند:

$ sudo apt-get install curl git mercurial make binutils bison gcc build-essential

برای نصب gvm دستور زیر را در خط فرمان اجرا کنید:

$ bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)

حال که gvm نصب شده است دستور زیر را اجرا کنید تا از صحت نصب اطمینان حاصل کنید:

$ gvm version

از نسخه 1.5، مترجم Go مستقل شده است، بدین معنا که تمام کد‌های آن با نسخه 1.4، بجای C نوشته شده است. بنابراین برای اینکه بتوانیم نسخه 1.5 را کامپایل کنیم نیاز به نسخه 1.4 داریم. با اجرای دستورات زیر Go را برای سیستم خود نصب می‌نمایید:

$ gvm install go1.4.3 --source=http://g.pdan.ir/golang/go.git
$ gvm use go1.4.3
$ gvm install go1.5.3 --source=http://g.pdan.ir/golang/go.git

و اگر مورد تحریم قرار نگرفته‌اید source=http://g.pdan.ir/golang/go.git-- را از انتهای دستورات حذف نمایید تا gvm از سرور اصلی گوگل برای شما نصب کند.

تعریف فضای‌ کاری

از آنجایی که تمامی کدنویسی در زبان Go تحت چهارچوب خاصی توصیه می‌شود، پیشنهاد شده حتما متغیر محیطی GOPATH به محیط کار (Workspace) اشاره کند و همه کد‌ها در آنجا قرار بگیرند. gvm در زمان نصب برای هر نسخه‌ محیطی جداگانه در نظر می‌گیرد. برای اینکه بدانید محیط کاری در کدام مسیر قرار دارد، کافی است دستور زیر را اجرا کنید:

$ echo $GOPATH

نصب محیط کد نویسی

ویرایشگر‌ها و IDE‌ های متعددی با قابلیت‌های متفاوت و مزیت‌ها و کمیت‌های قابل سنجش از Go پشتیبانی می‌کنند. مهم این است سلیقه شما چی بپسندد و با کدام بیشتر راحت باشید. لیستی از پلاگین‌ها و ویرایشگر‌ها برای این زبان را می‌توانید از IDEsAndTextEditorPlugins دریافت کنید. در این پُست سعی می‌شود چند مورد را معرفی کنیم.

ویرایشگر ATOM

سایت رسمی: atom.io

از خصوصیات این ویرایشگر می‌توان به کد باز بودن آن و پشتیبانی از زبان Go با افزونه go-plus اشاره کرد. این ویرایشگر توسط Github در سال ۲۰۱۴ رونمایی شد. برای نصب این ویرایشگر کافی است به صفحه انتشار نسخه‌ها مراجعه کنید و نسخه متناسب با سیستم خود را دریافت کنید.

برای این کار میتوانید از دستور wget استفاده کنید (این دستور نسخه ۱.۳.۳ را برای سیستم‌های ۶۴ بیت دبیان دانلود می‌کند) :

$ wget -c https://github.com/atom/atom/releases/download/v1.3.3/atom-amd64.deb

سپس می‌توانید با دستور زیر آن را نصب نمایید:

$ sudo dpkg -i atom-amd64.deb

حال ویرایشگر نصب شده است، با اجرا کردن دستور زیر در خط فرمان می‌توانید افزونه go-plus را نصب کنید.

$ apm install go-plus

ویرایشگر Sublime

سایت رسمی: sublimetext.org

سابلایم یک ویرایشگر کد محبوب در بین توسعه دهندگان است که متاسفانه آزاد نیست ولی بدون محدودیت می‌توانید می‌توانید از آن استفاده کنید تنها چند وقت یکبار پیامی آزار دهنده شما را تشویق به خریدن نسخه کامل می‌کند. از ویژگی‌های این ویرایشگر سادگی، سرعت اجرای بالا حتی در سیستم‌های ضعیف می‌باشد.

برای نصب ویرایشگر ابتدا آن را از سایت رسمی‌اش دریافت کنید و … اینطوری دریافت کنید:

$ wget -c https://download.sublimetext.com/sublime-text_build-3083_amd64.deb
$ sudo dpkg -i sublime-text_build-3083_amd64.deb
$ subl

با اجرای دستور آخر برنامه اجرا می‌شود و شما باید Package Control را که یک نصاب افزونه برای سابلایم است روی آن نصب نمایید. (مراجعه به سایت سازنده)

بعد از نصب Package Control و راه اندازی دوباره ویرایشگر، کلید‌های ترکیبی ctrl+shift+p را بفشارید و در جعبه متنی عبارت ‍Package Control: Install Package را وارد کنید تا لیست افزونه ها ظاهر شود. سپس افزونه GoSublime را از آن انتخاب کنید تا نصب گردد.

ساخت برنامه Hello World

در اینجا می‌خواهیم برنامه‌ای بسازیم که کلمه معروف Hello World را در خروجی چاپ نماید.

برای این کار باید دایرکتوری جدیدی در GOPATH/src$ ایجاد کنیم. بهتر آن است که این آدرس از روی منبع کد Github باشد (مثلا GOPATH/src/github.com/go-macaron/inject$ ) که با دستور go get برای استفاده دوباره هماهنگی داشته باشد، اما اینجا به منظور کوتاه کردن کلام و راحت تر ایجاد شدن اولین پروژه به این صورت عمل می‌کنیم:

  1. دستور mkdir $GOPATH/src/helloworld را در خط فرمان اجرا می‌کنیم.

  2. دستور subl $GOPATH/src/helloworld را به منظور باز شدن پروژه در ویرایشگر اجرا می‌کنیم.

  3. یک فایل جدید بنام helloworld.go در شاخه اصلی ایجاد می‌کنیم و محتوای زیر را در آن قرار می‌دهیم:

   package main

   import (
     "fmt"
   )

   func main() {
     fmt.Println("Hello World")
   }
  1. حال دستور go run $GOPATH/src/helloworld/helloworld.go را در خط فرمان اجرا کنید، خروجی را بصورت زیر مشاهده خواهید کرد.
